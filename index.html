Introduccion a HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Fue creado por Tim Berners-Lee en 1991 y ha evolucionado a lo largo de los años para incluir nuevas características y mejoras.

Introduccion a CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de un documento escrito en HTML. CSS permite a los desarrolladores separar el contenido de la presentación, lo que facilita la creación de diseños web atractivos y responsivos.

Introduccion a JavaScript

JavaScript es un lenguaje de programación de alto nivel, dinámico y de tipado débil. Es uno de los lenguajes de programación más populares y se utiliza principalmente para crear contenido interactivo en páginas web.
