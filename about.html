Introduccion a HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Fue creado por Tim Berners-Lee en 1991 y ha evolucionado a lo largo de los años para incluir nuevas características y mejoras.

Introduccion a CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento HTML. Fue desarrollado por Håkon Wium Lie en 1996 y permite a los desarrolladores web separar el contenido de la presentación, facilitando el diseño y la personalización de las páginas web.

Introduccion a JavaScript

JavaScript es un lenguaje de programación de alto nivel, dinámico y de tipado débil. Se utiliza principalmente para crear contenido web interactivo y dinámico. Fue creado por Brendan Eich en 1995 y se ha convertido en uno de los lenguajes de programación más populares del mundo.
